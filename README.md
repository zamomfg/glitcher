 # Script to create a glitch effect on faces in images

 ### Usage

positional arguments:
  img_path

options:
  -h, --help
  -o, --out
  -d, --debug


`python3 glitcher.py <filename>`
`python3 glitcher.py <filename> -o <output file>`
